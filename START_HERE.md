## ✅ מה כבר נבנה (שלב 3 + תוספות)

המבנה: **קורס → פרקים (לחיצים) → כל פרק: חומר לימוד + מבחן עם שאלות הפרק.**

1. **ניהול פרקים** — הוספה/עריכה/מחיקה/סידור (↑↓). דף `AdminChaptersPage`.
2. **שאלות לפי פרק** (לא לפי קורס) — 3 סוגים:
   - אמריקאית (תשובה אחת)
   - בחירה מרובה (ניקוד חלקי)
   - שאלה פתוחה (בדיקה ידנית — ציון זמני מיידי, השלמה אחרי בדיקה)
   - אפשר להוסיף וידאו לשאלה
3. **חומר לימוד לכל פרק** (לא רק מבוא לקורס) + **סוג "טקסט + תמונה" (RICH)**.
4. **עריכת חומר לימוד** (קודם אפשר היה רק למחוק).
5. **תפריט צד (Sidebar)** — רשימת כל הקורסים, לחיצה פותחת פרקים, ניווט מכל דף תלמיד. מותאם לנייד (כפתור ☰).
6. **תיקון באג מחיקת קורס** — היה כשל בגלל FK; תוקן (CASCADE).
