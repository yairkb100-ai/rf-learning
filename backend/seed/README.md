# סקריפטי Seed — תוכן קורסים ושרטוטים

סקריפטים חד-פעמיים שיצרו את תוכן הקורסים, השאלות והשרטוטים ב-DB.
**התוכן כבר קיים ב-DB** — אין צורך להריץ אותם בהפעלה רגילה.

כולם **אידמפוטנטיים** (UPSERT לפי כותרת/שם) — בטוח להריץ שוב כדי לעדכן/לתקן תוכן.

הרצה (מתוך תיקייה זו):
```bash
node <script>.js
```

| קובץ | מה הוא עושה |
|------|-------------|
| `gen_engine_diagrams.js` | יוצר שרטוטי SVG של מנועים ב-`../uploads/` |
| `gen_arch_diagrams.js` | יוצר שרטוטי SVG של ארכיטקטורת מחשבים ב-`../uploads/` |
| `add_deep_content.js` | תוכן עיוני מעמיק לפרקי המנועים |
| `rewrite_intro_content.js` | שכתוב תוכן המבוא של פרקי המנועים (פורמט נקי) |
| `tidy_engine_images.js` | משייך תמונה אחת לכל פרק מנוע |
| `add_engine_parts.js` | פרק "חלקי המנוע ותפקידם" + תוכן |
| `add_engine_questions.js` | שאלות לפרקי המנועים |
| `add_architecture_course.js` | מגמת "מחשוב ובקרה" + קורס "ארכיטקטורת מחשבים" + תוכן |
