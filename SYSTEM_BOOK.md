### 👨‍🎓 כמה תלמיד משתמש בלומדה

#### צעד 1: התחברות
1. הזן ת"ז ו סיסמה

#### צעד 2: בחירת קורס
בדף הבית תראה **רשימה של קורסים**.

כל קורס מציג:
- **שם הקורס**
- **תיאור קצר**
- **כפתור "התחל"** להיכנס לקורס

**לחץ על "התחל"** בקורס שאתה רוצה.

#### צעד 3: צפייה בפרקים
בדף הקורס תראה:

**עמודה שמאלית:**
- רשימת כל פרקי הקורס
- כל פרק מציג:
  - 📖 שם הפרק
  - ✅ סטטוס (סיים / בתהליך)
  - 📊 התקדמות

**עמודה ימנית (sticky card):**
- אחוז ההשלמה של הקורס
- מספר פרקים שהושלמו
- כפתור "המשך לקורס"

#### צעד 4: קריאת פרק
**לחץ על פרק** מהרשימה.

בדף הפרק תראה:
- **כותרת הפרק**
- **תיאור מלא** (עם עיצוב: **בולד**, *אטליק*, <u>קו תחתון</u>, רשימות, וכו')
- **תוכן חינוכי** (טקסט + תמונות + קבצים)
- **קבצים להורדה** (PDF, Word, Excel, וכו')

#### צעד 5: פתרון בחינות
אם יש בחינות בקורס:
1. **לחץ על "בחינות"** בתפריט
2. **בחר בחינה**
3. **ענה על שאלות** (רשות רבת-ברירה / בחירה מרובה / טקסט חופשי)
4. **שלח תשובות**
5. **קבל ציון**

3. **בחר סוג תוכן:**
   - 📄 **טקסט עשיר** (text + עיצוב) - משתמש בעורך טקסט עשיר
   - 📷 **תמונה** - upload תמונה
   - 📑 **קובץ** (PDF, Word, Excel, PowerPoint)

#### **אם בחרת "טקסט עשיר":**
1. **כתוב את הטקסט** בעורך

   - **B** = בולד
   - **I** = אטליק
   - **U** = קו תחתון

#### **אם בחרת "קובץ":**
1. **לחץ "בחר קובץ"**

3. **לחץ "שמור"**

✅ **התוכן הוסף בהצלחה!**

## עורך הטקסט העשיר

### מה זה עורך טקסט עשיר (WYSIWYG)?

**עורך טקסט עשיר** מאפשר לך לעצב טקסט **ישירות**, בדומה ל-Word.

### כלי הסרגל

| כלי | תיאור | דוגמה |
|-----|-------|--------|
| **B** | בולד | **טקסט כזה** |
| **I** | אטליק | *טקסט כזה* |
| **U** | קו תחתון | <u>טקסט כזה</u> |
| **H** | כותרת (Heading 3) | ### כותרת |
| **🎨** | בחר צבע | <span style="color:red">טקסט צבוע</span> |
| **≡** | רשימה לא מסודרת | • פריט 1<br>• פריט 2 |
| **1.** | רשימה מסודרת | 1. פריט 1<br>2. פריט 2 |
| **<** | יישור לשמאל | ← יישור |
| **>** | יישור לימין | יישור → |
| **||** | יישור למרכז | → יישור ← |
| **X** | ניקוי עיצוב | טקסט רגיל |

### דוגמה: כתיבת תוכן מפורט

```
מנוע בוכנה - עיקרון הפעולה

(כותרת - לחץ H)

מנוע בוכנה הוא מנוע המשתמש בבוכנות לייצור אנרגיה.

יתרונות:
• עלות נמוכה
• עמידות גבוהה
• דלק זול

חסרונות:
• זיהום אוויר גבוה
• קול רועם

(רשימה לא מסודרת - לחץ על ≡ ובחר "Unordered List")
```

### ❓ "איך יוצרים בחינה?"

**תיאור קצר:**
1. בפאנל ניהול, בחר "שאלות"
2. הוסף שאלה חדשה
3. בחר סוג: רשות רבת-ברירה / בחירה מרובה / טקסט חופשי
4. שמור

### ❓ "הלוגו בראש העמוד לא קליק"

לחץ על "לומדת ענף 71" בראש כל עמוד כדי לחזור לדף הבית.

### ❓ "איך מחוקים קורס?"

**כן!**
1. בפאנל ניהול, בחר "קורסים"
2. בחר את הקורס
